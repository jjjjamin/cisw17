What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and design web pages.

What is CSS?

CSS (Cascading Style Sheets) is used to style and format the layout of web pages, like colors, fonts, and spacing.

How do I create a link in HTML?

Use the <a> tag with the href attribute: <a> href="URL">Link Text</a> .

What is an internal link in HTML?

An internal link connects to another page or section within the same website using relative URLs.

What is the difference between an ID and a class in CSS?

IDs are unique and apply to a single element (#id), while classes can be reused for multiple elements (.class).

How do I center text in CSS?

Use the text-align: center; property for text elements.

What is the box model in CSS?

The box model describes the layout of elements: content, padding, border, and margin.

How do you link a CSS file to an HTML file?

Use the <link> tag in the HTML <head>: <link rel="stylesheet" href="styles.css">.

What is responsive design in CSS?

Responsive design makes web pages adapt to different screen sizes using media queries like @media.

What is the purpose of the <div> element?

The <div> element is used to group content and apply CSS styling to multiple elements.
