Lesson 7: Using External and Internal Links

One of the most important components of our web page are the links displayed throughout them. These links can be used to redirect someone to multiple paths, internally and externally. To create a link within your web page, you have to use the <a> tag, making the hypertext "hyper." With that you can set the link to different parts of your page, known as anchor points. We also learned that when setting these links for other web pages, we have to also include "href" within the <a> tag. If we wanted to link within our own page, we would just include the filename within the directory.

There are other uses and things we can link with the <a> tag. We can allow people visiting our web page to have a direct link to email us, even customizing it to include a premade subject line and body text. We can link direct downloads to different files, including .pdf and .png. We can even have an image set up to be the actual link within the page. One of the biggest takeaways is to have our links stand out when compared to the rest of the body text. We need to have different ways to signify their importance, from changing their color, making them bold, or even underlining them. We can also add more effects like a hover and visited state.
